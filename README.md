# 푸리에·신호처리, 눈으로 보기 (Fourier & Signal Processing, Visualized)

세상의 모든 신호는 **단순한 물결(사인파)의 합** — 회전·파형·스펙트럼·에피사이클을 직접 돌리고 **소리로 들으며** 익히는 인터랙티브 학습 자료입니다. 모든 신호·변환·소리가 브라우저에서 실시간으로 계산됩니다. 복소수·적분 기호 없이, 끝까지 회전하는 화살표와 물결로만 풀어냈어요.

> 직접 만든 **비영리 학습용 자료**입니다. 특정 교재·강의와 무관하며, 설명·예시·코드는 모두 독자적으로 제작했습니다.

## 시작하기

빌드·의존성 없음. `index.html` 을 브라우저로 열면 끝입니다. (🔊는 브라우저 정책상 버튼 클릭 후 재생됩니다.)

```bash
open index.html                 # macOS
# 또는 간단한 로컬 서버
python3 -m http.server 8000     # → http://localhost:8000
```

## 설계 원칙

- **실시간 계산**: 합성·분해·DFT/FFT·필터·스펙트로그램이 모두 실제 코드로 동작 (하드코딩 결과 없음)
- **수식·복소수 없는 직관 우선**: 회전 화살표와 물결로 — 일부 챕터는 🔊 실제 소리로 시각+청각 학습
- **단계별 분해**: 각 챕터 4스텝, 비유 → 관찰(▶재생) → 조작(슬라이더/🔊) → 정리(+실무 연결)
