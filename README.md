# 푸리에·신호처리, 눈으로 보기 (Fourier & Signal Processing, Visualized)

세상의 모든 신호는 **단순한 물결(사인파)의 합** — 회전·파형·스펙트럼·에피사이클을 직접 돌리고 **소리로 들으며** 익히는 인터랙티브 학습 자료입니다. 모든 신호·변환·소리가 브라우저에서 실시간으로 계산됩니다. 복소수·적분 기호 없이, 끝까지 **회전하는 화살표와 물결**로만 풀어냈어요.

> 직접 만든 **비영리 학습용 자료**입니다. 특정 교재·강의와 무관하며, 설명·예시·코드는 모두 독자적으로 제작했습니다.

## 핵심 — 회전·소리·선대

- **회전하는 화살표(위상자)**가 모든 챕터의 주인공. 복소수는 "도는 화살표"로 끝까지 우회.
- **에피사이클**(CH07): 회전 화살표 수십 개가 이어져 하트를 그려내는 장면.
- **🔊 Web Audio**: 합성·필터·압축을 시각 + 청각으로(CH04·13·17).
- **선형대수 연결**(CH15~17): 푸리에 계수 = 사인 기저로의 **사영(내적)**, 사인파끼리 **직교**, 변환 = **기저 바꾸기**.

**PART 4 · 현실의 신호** — 12 샘플링·에일리어싱 · 13 필터링(🔊) · 14 스펙트로그램
**PART 5 · 선대의 눈 + 응용** — 15 사영=계수(내적) · 16 직교 기저 · 17 기저 변환·압축(🔊)

### VZ.FT 엔진 — 등속 시간 클럭이 심장

회전·파형·스펙트럼을 같은 시각 `t`로 동기화하는 `clock`(무한 애니메이션 루프) 위에 렌더러를 얹습니다: `cboard`(복소평면) · `phasor`(회전 화살표+그림자) · `epicycles`+`trail`(궤적) · `waveStrip`(파형) · `spectrum`(주파수 막대) · `dft`/`fft`/`dftComplex` · `projectionArea`(사영=내적) · `heatColor`(스펙트로그램) · `audio`(Web Audio 합성).

## 설계 원칙

- **실시간 계산**: 합성·분해·DFT/FFT·필터·스펙트로그램이 모두 실제 코드로 동작 (하드코딩 결과 없음, 수치 검증 완료)
- **수식·복소수 없는 직관 우선**: 회전 화살표와 물결로 — 기호는 등장할 때마다 풀이
- **단계별 분해**: 각 챕터 4스텝, 비유 → 관찰(▶재생) → 조작(슬라이더/🔊) → 정리(+실무 연결)
